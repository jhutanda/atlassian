# AcademiChain AI Automator - Product Overview

## Purpose
An Atlassian Forge app that automates academic workflows within educational institutions using Jira, Confluence, and Jira Service Management.

## Core Features
- **Assignment Management**: Create, track, and grade student assignments with automated workflows
- **Project Proposal System**: Multi-stage approval workflow for student project proposals (Faculty → HOD → Guide Assignment)
- **Semester Management**: Confluence-based semester organization with lecture notes and attendance tracking
- **Analytics Dashboard**: Real-time statistics on assignments, submissions, and project approvals

## Target Users
- **Faculty**: Create assignments, manage courses, grade submissions, approve project proposals
- **Students**: Submit assignments, propose projects, track academic progress
- **Administrators**: View analytics, configure academic templates, manage user roles

## Key Integrations
- **Jira**: Custom issue types for assignments and project proposals with academic-specific custom fields
- **Confluence**: Semester spaces, assignment feedback pages, attendance tracking macros
- **JSM**: Project proposal approval workflows with automated routing
- **Forge Storage**: Academic configuration and workflow templates

## Business Value
Streamlines academic processes by leveraging Atlassian's collaboration platform, reducing manual overhead for faculty and providing students with clear visibility into their academic progress.
